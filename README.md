- Hints

- 3 difficulty levels (easy, normal and hard)

### Implemented in the code
1. Input error handling (if the user entered text instead of a number).
2. Distance hints
3. Remembering past attempts
4. The choice of difficulty
5. After finishing the game, offer to play again without restarting the program.
6. Saving the results
7. Using colors in the terminal
